# dotfiles by jteng2127

## Install

clone this repo into `~/.dotfiles` ~~or other place you want~~
install stow
and install module you want by

```bash
cd ~/.dotfiles
stow --no-folding <module name>
```

all modules are folder names in `~/.dotfiles`

## Uninstall

```bash
cd ~/.dotfiles
stow -D <module name>
```
